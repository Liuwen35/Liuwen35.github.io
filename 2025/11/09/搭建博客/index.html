搭建博客

一.电脑系统环境配置

1.安装下载git

2.安装下载Node.js

（安装完毕可以用git输入npm -v或npm -v执行输出版本号进行验证是否成功安装）

二.安装HEXO

1.全局安装Hexo命令行工具
在Git Bash里执行npm install hexo-cli -g
(安装完毕可以在git输入执行hexo -v输出版本号进行验证是否成功安装)


2.初始化Hexo博客项目
决定好存放blog文件的目录，在该目录打开git输入执行hexo init blog创建Hexo初始模板

三.GitHub Pages设置

1.注册账号
访问http://www.github.com/注册GitHub账号(username和邮箱非常重要！！！)
2.创建账号仓库
仓库名为 username.github.io,配置可见性为公开

3.Git全局身份配置
先在git输入ls -al ~/.ssh查看本地是否存在ssh key,若存在直接全部删除。然后输入git config –global user.name ““和git config –global user.email ““(这里的username和email优先与GitHub账号一致哦)
4.生成shh密钥
输入ssh-keygen -t rsa -C ““(此处的邮箱为注册GitHub账号时使用的)，后续一路按回车键即可
5.获取ssh密钥
输入执行cat ~/.ssh/id_rsa.pub输出密钥，后复制密钥(整个输出)，在GitHub打开设置找到SSH and GPG keys进行编辑
(完成后执行ssh git@github.com查看密钥配对情况)

四.部署到GitHub

1.配置_config.yml
编辑blog文件中的_config.yml，修改Deployment部分
(deploy:
type: git
repo: ssh://git@github.com/username/username.github.io(你的仓库)
branch: main)【每一个：后面都有一个空格】
2.安装Git部署插件
输入执行npm install hexo-deployer-git –save

3.博客部署
输入执行hexo deploy

ok,如果顺利的话到这一步你就成功了！！！
